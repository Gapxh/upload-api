## How to Install

### Need Node.js to run this project

- Fork the repo
- Clone repo

```
git clone https://github.com/H3X-T/upload-api/
```

- Install package

```
npm install || yarn install
```

- Edit example.env (remove exmaple to .env)

```
PORT=
DBURL=
CLOUND_NAME=
CLOUND_API_KEY=
CLOUND_API_SECRET=
```

- Start

```
npm run dev || yarn run dewv
```
